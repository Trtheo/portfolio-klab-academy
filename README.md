# Portfolio Website

A modern, responsive portfolio website built with HTML, CSS.

## Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Fixed Navigation** - Sticky header with smooth scrolling
- **Mobile Menu** - Professional hamburger menu for mobile devices
- **Interactive Sections** - Portfolio modal, contact form, blog pagination
- **Modern UI** - Purple gradient theme with glassmorphism effects

## Sections

- **Hero** - Introduction with profile photo and statistics
- **About** - Personal information and social links
- **Process** - 4-step work process overview
- **Portfolio** - Project showcase with modal details
- **Blog** - Recent posts with pagination
- **Services** - UX/UI design and development services
- **Contact** - Contact form and information
- **Footer** - Navigation links and copyright

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript
- Font Awesome Icons

## Setup

1. Clone or download the repository
2. Replace `photo.jpg` with your actual photo
3. Update personal information in `index.html`
4. Customize colors and content as needed
5. Open `index.html` in a web browser

## File Structure

```
portfolio-2025/
├── index.html          # Main HTML file
├── styles.css          # CSS styles
├── your-photo.jpg      # Profile photo (add your own)
├── 1.png - 7.png      # Design reference images
└── README.md          # This file
```

## Customization

- **Colors**: Update CSS variables for purple gradient theme
- **Content**: Modify text content in HTML sections
- **Images**: Replace placeholder images with your projects
- **Contact**: Update contact information and form action
